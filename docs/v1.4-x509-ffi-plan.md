# v1.4 — C FFI for X.509-with-SM2 — Implementation Plan

> **For agentic workers:** execute task-by-task, TDD, frequent commits.
> Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Expose the v1.3 `gmcrypto_core::x509::Certificate` surface through
the `gmcrypto-c` C ABI — 13 always-on symbols + 1 opaque handle + 1 plain
time struct (72 → 85 entry points), per `docs/v1.4-scope.md` Q4.1–Q4.15.

**Architecture:** thin shim, zero new crypto. One immutable opaque handle
(`gmcrypto_x509_certificate_t`) wrapping the Rust `Certificate`; copy-out
accessors via the existing `write_output` helper; issuer keys and the
exported subject key ride the existing `gmcrypto_sm2_pubkey_t` handle; ID
defaulting rides the existing `kx_id` helper. Single `GMCRYPTO_ERR`/NULL
everywhere.

**Tech stack:** Rust 2024 / MSRV 1.85, cbindgen 0.29 (`regen-header`),
c_smoke `extern "C"` Rust-equivalence tests, cargo-fuzz (`fuzz_c_abi` op
extension), gmssl 3.1.1 fixtures already committed under
`crates/gmcrypto-core/tests/data/`.

```rust
fn copy_out(
    f: unsafe extern "C" fn(
        *const gmcrypto_x509_certificate_t,
        *mut u8,
        usize,
        *mut usize,
    ) -> core::ffi::c_int,
    cert: *const gmcrypto_x509_certificate_t,
) -> Vec<u8> {
    let mut needed = 0usize;
    // Length discovery: undersized (zero) capacity reports required length.
    unsafe { f(cert, std::ptr::null_mut(), 0, &mut needed) };
    let mut buf = vec![0u8; needed.max(1)];
    let mut actual = 0usize;
    let rc = unsafe { f(cert, buf.as_mut_ptr(), buf.len(), &mut actual) };
    assert_eq!(rc, GMCRYPTO_OK);
    buf.truncate(actual);
    buf
}

/// Q4.6 (review D2): surgically remove the `[3]` extensions block from the
/// leaf and patch the two enclosing SEQUENCE lengths. The now-invalid
/// signature is irrelevant — `from_der` never verifies.
fn leaf_without_extensions() -> Vec<u8> {
    let ext_seq = Certificate::from_der(X509_LEAF_DER).unwrap();
    let ext_seq = ext_seq.extensions_raw().unwrap();
    let pos = X509_LEAF_DER
        .windows(ext_seq.len())
        .position(|w| w == ext_seq)
        .unwrap();
    // The [3] EXPLICIT wrapper (0xA3 + DER length) directly precedes it.
    let wrap_start = if X509_LEAF_DER[pos - 2] == 0xA3 {
        pos - 2
    } else if X509_LEAF_DER[pos - 3] == 0xA3 && X509_LEAF_DER[pos - 2] == 0x81 {
        pos - 3
    } else {
        assert_eq!(X509_LEAF_DER[pos - 4], 0xA3, "unexpected [3] length form");
        pos - 4
    };
    let removed = (pos + ext_seq.len()) - wrap_start;
    let mut t = X509_LEAF_DER[..wrap_start].to_vec();
    t.extend_from_slice(&X509_LEAF_DER[pos + ext_seq.len()..]);
    // Patch the outer Certificate SEQUENCE and tbs SEQUENCE lengths (both
    // long-form 0x82 for these ~1 KiB fixtures; asserted).
    assert_eq!((t[0], t[1]), (0x30, 0x82));
    assert_eq!((t[4], t[5]), (0x30, 0x82));
    for len_at in [2usize, 6] {
        let old = (usize::from(t[len_at]) << 8) | usize::from(t[len_at + 1]);
        let new = old - removed;
        t[len_at] = u8::try_from(new >> 8).unwrap();
        t[len_at + 1] = u8::try_from(new & 0xff).unwrap();
    }
    t
}
```

```rust
// ============================================================
// X.509-with-SM2 certificate parse + signature verify (v1.4).
// ============================================================

/// Opaque X.509 certificate handle (v1.4). Immutable after construction —
/// every accessor takes `const *`; free with
/// [`gmcrypto_x509_certificate_free`].
pub struct gmcrypto_x509_certificate_t {
    inner: Certificate,
}

/// Parse a DER X.509-with-SM2 **v3 leaf** certificate
/// (`docs/v1.3-x509-sm2-design.md` profile). Returns NULL on ANY failure
/// (null/empty input, malformed DER, wrong profile) — indistinguishable by
/// design. **Parsing makes NO trust decisions** (no chain / time /
/// extension / revocation logic — see `gmcrypto-core`'s `x509` module).
///
/// # Safety
/// `der` must be valid for `der_len` reads (or `der_len == 0`).
#[unsafe(no_mangle)]
pub unsafe extern "C" fn gmcrypto_x509_certificate_from_der(
    der: *const u8,
    der_len: usize,
) -> *mut gmcrypto_x509_certificate_t {
    let result = std::panic::catch_unwind(|| {
        let bytes = unsafe { try_slice(der, der_len) }?;
        let inner = Certificate::from_der(bytes)?;
        Some(Box::into_raw(Box::new(gmcrypto_x509_certificate_t { inner })))
    });
    match result {
        Ok(Some(p)) => p,
        _ => ptr::null_mut(),
    }
}

/// Free a certificate handle. NULL is a no-op.
///
/// # Safety
/// `cert` must be a handle from this library, not yet freed.
#[unsafe(no_mangle)]
pub unsafe extern "C" fn gmcrypto_x509_certificate_free(
    cert: *mut gmcrypto_x509_certificate_t,
) {
    if cert.is_null() {
        return;
    }
    drop(unsafe { Box::from_raw(cert) });
}
```

```rust
/// Shared copy-out body for the raw-byte accessors.
///
/// NOTE (review A1): the closure MUST be `move` — a by-reference capture of
/// the generic `get` makes the closure `&impl Fn…`, which fails `ffi_guard`'s
/// `UnwindSafe` bound (`&G: UnwindSafe` needs `G: RefUnwindSafe`); the `move`
/// capture-by-value makes the declared `+ UnwindSafe` bound suffice
/// (empirically verified by the reviewer).
///
/// # Safety
/// Forwarded caller contract — `out` valid for `out_capacity` writes,
/// `out_actual_len` valid for one write. This is the first `write_output`
/// site whose source slice borrows from the handle itself (every prior site
/// copies from a locally-owned buffer); a caller passing `out` overlapping
/// the handle's interior heap would be `&`/`&mut` aliasing — unreachable
/// without already-UB pointer forgery, since no API exposes those addresses
/// (review B2).
unsafe fn x509_copy_out(
    cert: *const gmcrypto_x509_certificate_t,
    get: impl Fn(&Certificate) -> &[u8] + std::panic::UnwindSafe,
    out: *mut u8,
    out_capacity: usize,
    out_actual_len: *mut usize,
) -> c_int {
    ffi_guard(move || {
        let Some(cert) = (unsafe { cert.as_ref() }) else {
            return GMCRYPTO_ERR;
        };
        unsafe { write_output(get(&cert.inner), out, out_capacity, out_actual_len) }
    })
}

/// The exact `TBSCertificate` TLV bytes (the bytes the signature covers),
/// copy-out convention: too-small buffer → `GMCRYPTO_ERR` + required length
/// in `*out_actual_len`.
///
/// # Safety
/// `cert` valid handle; `out` valid for `out_capacity` writes;
/// `out_actual_len` valid for one write.
#[unsafe(no_mangle)]
pub unsafe extern "C" fn gmcrypto_x509_certificate_tbs_raw(
    cert: *const gmcrypto_x509_certificate_t,
    out: *mut u8,
    out_capacity: usize,
    out_actual_len: *mut usize,
) -> c_int {
    unsafe { x509_copy_out(cert, |c| c.tbs_raw(), out, out_capacity, out_actual_len) }
}
```

```rust
/// Verify the certificate's SM2-with-SM3 signature over the exact wire
/// `tbsCertificate` bytes against `issuer`, using the GM / RFC 8998
/// default signer ID `"1234567812345678"`. Returns [`GMCRYPTO_OK`] only on
/// a verifying signature; [`GMCRYPTO_ERR`] on invalid or any error
/// (indistinguishable by design).
///
/// **This is NOT certificate validation** — no chain / time / extension /
/// revocation logic; `GMCRYPTO_OK` means exactly "this issuer key signed
/// these tbs bytes".
///
/// # Safety
/// `cert` / `issuer` must be valid handles from this library.
#[unsafe(no_mangle)]
pub unsafe extern "C" fn gmcrypto_x509_certificate_verify_signature(
    cert: *const gmcrypto_x509_certificate_t,
    issuer: *const gmcrypto_sm2_pubkey_t,
) -> c_int {
    unsafe {
        gmcrypto_x509_certificate_verify_signature_with_id(cert, issuer, ptr::null(), 0)
    }
}

/// As [`gmcrypto_x509_certificate_verify_signature`] with a caller-supplied
/// SM2 signer ID; `id_len == 0` selects the default ID (the v1.2 KX
/// convention — an empty ID is unrepresentable through this ABI).
///
/// # Safety
/// `cert` / `issuer` valid handles; `id` valid for `id_len` reads when
/// `id_len > 0`.
#[unsafe(no_mangle)]
pub unsafe extern "C" fn gmcrypto_x509_certificate_verify_signature_with_id(
    cert: *const gmcrypto_x509_certificate_t,
    issuer: *const gmcrypto_sm2_pubkey_t,
    id: *const u8,
    id_len: usize,
) -> c_int {
    ffi_guard(|| {
        let (Some(cert), Some(issuer)) =
            (unsafe { cert.as_ref() }, unsafe { issuer.as_ref() })
        else {
            return GMCRYPTO_ERR;
        };
        // `kx_id`: id_len == 0 → DEFAULT_SIGNER_ID (shared v1.2 helper).
        let Some(id) = (unsafe { kx_id(id, id_len) }) else {
            return GMCRYPTO_ERR;
        };
        if cert.inner.verify_signature_with_id(&issuer.inner, id) {
            GMCRYPTO_OK
        } else {
            GMCRYPTO_ERR
        }
    })
}
```

# X.509-with-SM2 (v1.3) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans
> to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax.
> The companion spec `docs/v1.3-x509-sm2-design.md` (+ `docs/v1.3-scope.md`
> Q3.1–Q3.11) OVERRIDES this plan where they conflict.

**Goal:** Leaf X.509 certificate parse + SM2-with-SM3 signature verify behind
the opt-in `x509` feature — no trust decisions, no new dependency, no new
cryptographic code.

**Architecture:** One new module `crates/gmcrypto-core/src/x509.rs` composing
the existing strict `asn1::reader`, `spki::decode`, and `sm2::verify_with_id`.
`Certificate` owns the input bytes (`Vec<u8>`) plus index ranges into them —
no borrowed lifetimes in the public API (the pem/pkcs8 house style).

**Tech stack:** pure `no_std + alloc` Rust, existing workspace only.

**Fixture-first ordering (the v1.1 lesson):** real gmssl-generated KAT
certificates are captured in Task 1, BEFORE any parser code, so every parser
test runs against authentic GM/T 0015-profile bytes from day one.

```rust
/// Calendar timestamp from UTCTime/GeneralizedTime (Zulu only). The caller
/// owns any "currently valid" decision — this library has no clock.
#[derive(Clone, Copy, Debug, PartialEq, Eq, PartialOrd, Ord)]
pub struct X509Time {
    pub year: u16, pub month: u8, pub day: u8,
    pub hour: u8, pub minute: u8, pub second: u8,
}

const TAG_UTC_TIME: u8 = 0x17;
const TAG_GENERALIZED_TIME: u8 = 0x18;

fn two_digits(b: &[u8]) -> Option<u16> {
    match b {
        [a @ b'0'..=b'9', c @ b'0'..=b'9'] => Some(u16::from((a - b'0') * 10 + (c - b'0'))),
        _ => None,
    }
}

/// Parse one Time value (either tag), returning (time, rest_after_tlv).
fn read_time(input: &[u8]) -> Option<(X509Time, &[u8])> {
    let (year, body, rest) = if let Some((v, rest)) = reader::read_tlv(input, TAG_UTC_TIME) {
        if v.len() != 13 { return None; }
        let yy = two_digits(&v[0..2])?;
        (if yy >= 50 { 1900 + yy } else { 2000 + yy }, &v[2..], rest)
    } else {
        let (v, rest) = reader::read_tlv(input, TAG_GENERALIZED_TIME)?;
        if v.len() != 15 { return None; }
        (two_digits(&v[0..2])? * 100 + two_digits(&v[2..4])?, &v[4..], rest)
    };
    // body = MMDDHHMMSS + 'Z'
    if body.len() != 11 || body[10] != b'Z' { return None; }
    let (month, day) = (two_digits(&body[0..2])? as u8, two_digits(&body[2..4])? as u8);
    let (hour, minute, second) = (two_digits(&body[4..6])? as u8,
                                  two_digits(&body[6..8])? as u8,
                                  two_digits(&body[8..10])? as u8);
    if !(1..=12).contains(&month) || !(1..=31).contains(&day)
        || hour > 23 || minute > 59 || second > 59 { return None; }
    Some((X509Time { year, month, day, hour, minute, second }, rest))
}
```

```rust
/// Read an AlgorithmIdentifier that MUST be sm2-sign-with-sm3 with absent or
/// NULL parameters (design Q3.6). Returns (full_algid_tlv_span, rest).
fn read_sm2_sig_algid(input: &[u8]) -> Option<(&[u8], &[u8])> {
    let (body, rest) = reader::read_sequence(input)?;
    let span = &input[..input.len() - rest.len()];
    let (oid, after_oid) = reader::read_oid(body)?;
    if oid != oid::SM2_SIGN_WITH_SM3 {
        return None;
    }
    match after_oid {
        [] => Some((span, rest)),
        _ => {
            let after_null = reader::read_null(after_oid)?;
            if after_null.is_empty() { Some((span, rest)) } else { None }
        }
    }
}
```

# v1.7 — TLCP record protection — Implementation Plan

> **For agentic workers:** Execute task-by-task, TDD (red → green → commit).
> Steps use checkbox (`- [ ]`) syntax. This plan implements the
> maintainer-signed `docs/v1.7-scope.md` (Q7.1–Q7.11) and
> `docs/v1.7-kat-sourcing.md`. The hard constant-time code (Stage B) is
> written against the **`ct_tlcp_cbc_deprotect` dudect target built FIRST**
> (Task C1 is pulled earlier — see Stage B note); the reference code below
> is the design, refined in the red-green loop until `|tau| < 0.20`.

**Goal:** Ship the TLCP (TLS-1.1-with-SM) record protect/deprotect
primitives for the four SM2-family suites — SM4-GCM (RFC 5288 shape) and
SM4-CBC (MAC-then-encrypt, TLS padding, **Lucky13-class constant-time
deprotect**) — behind the existing opt-in `tlcp` umbrella feature, as a new
`tlcp::record` module.

**Architecture:** Engine-shaped, byte-in/byte-out, no hidden state.
Per-direction `ZeroizeOnDrop` keys carriers carved from `derive_key_block`;
explicit caller-held `seq: u64`; injected RNG for the CBC explicit IV. GCM
is a thin nonce/AAD-shaping wrapper over the existing `mode_gcm`. CBC composes
raw block-CBC over `Sm4Cipher::{encrypt,decrypt}_block` + a **new
`pub(crate)` SM3 capability** (widen `compress` + a clone-and-discard
dummy-compress) for the constant-compression-count MAC. Single failure mode
(`Option`), no plaintext on failure.

**Tech Stack:** Rust 2024, MSRV 1.85, `no_std` + `alloc`, `unsafe_code =
forbid`. Deps already present: `subtle`, `zeroize`, `rand_core`
(`TryCryptoRng`), `core::hint::black_box`. NO new dependency.

**Branch:** `feat/tlcp-record-protection`. Workspace `1.6.0 → 1.7.0`,
sibling pins `=1.7.0` (Q7.9). Publish stays the maintainer's call.

## W2 review corrections (Fable-5 adversarial review, EXECUTED — apply during W3)

A 3-agent panel reviewed this plan; one agent **compiled + ran** the
load-bearing Lucky13 core in a throwaway worktree. **Verdict: GO-WITH-FIXES,
`ready_for_implementation: true`.** Execution confirmed the design is sound
(equalized MAC == ordinary `HmacSm3` for lengths {0,1,13,42,43,44,55,56,57,100,256,1000};
the total-compress-count equalization invariant HOLDS for every secret `P ≤ max`;
`inner_blocks` is byte-exact vs SM3's spill rule incl. the 55/56/57 boundary;
`extract_mac_ct` is CT-feasible; CBC round-trips + rejects bad MAC/pad/seq with
no panics). The corrections below **supersede the inline code at the cited
steps** — apply them.

**MF1 — GCM record path needs `sm4-aead`.** `sm4::mode_gcm` is
`#[cfg(feature="sm4-aead")]` (sm4/mod.rs:35) and `tlcp = []` does not enable
it, so the Stage-A GCM fns **do not compile under `--features tlcp` alone**
(reproduced: `E0433: cannot find mode_gcm`). Cfg-gate `RecordKeysGcm` +
`gcm_nonce` + `protect_gcm`/`deprotect_gcm` + their `use mode_gcm` on
`#[cfg(feature = "sm4-aead")]` (CBC stays pure-core under `tlcp`; GCM callers
enable `tlcp,sm4-aead`). **Add a `tlcp,sm4-aead` leg to the C4 clippy/deny/
wasm/dudect matrix.** Do NOT use `tlcp = ["sm4-aead"]` — it pulls the
`gmcrypto-simd` GHASH dep and breaks the wasm32 `--no-default-features
--features tlcp` leg, contradicting the no-new-dep posture. *(This is a
user-facing feature-gating consequence flagged for the maintainer: GCM record
protection requires `tlcp,sm4-aead`; CBC record protection is pure `tlcp`.)*

**MF2 — `usize::conditional_select` does not exist** (`subtle` impls
`ConditionallySelectable` only for fixed-width ints; reproduced `E0599`). At
both sites (`check_tls_padding_ct`, `deprotect_cbc`) select on `u64` and cast:
`u64::conditional_select(&(a as u64), &(b as u64), c) as usize`
(+ `#[allow(clippy::cast_possible_truncation)]`).

**MF3 — `deprotect_cbc` 2^14 ceiling (a Lucky13 chain-break, not just DoS).**
Without an upper bound, a ~64 KB record makes `u16::try_from(eff_len).ok()?` a
**secret-dependent early return that SKIPS the MAC** on bad-pad records
(`eff_len = max_plaintext_len > 65535`), and `(n − 32) as u16` truncates. Add,
as the FIRST guard (public length, before any secret arithmetic), mirroring
`deprotect_gcm`:
```rust
if body_len > (1 << 14) + 32 + 16 { return None; }   // 2^14 pt + 32 MAC + 16 pad
```
This bounds `n` so `(n − 32)` fits u16 losslessly AND `eff_len` always fits u16
by construction — then derive the MAC-header length with **NO fallible `?` on
the secret** (cast the now-bounded `eff_len as u16`). The header `length` still
carries the secret `eff_len` for interop (Q7.3); only the fallible conversion
is removed.

**MF4 — dudect C1 axis is wrong (false assurance).** Task C1 as drafted
class-splits by KEY with both classes the SAME fragment length → the Lucky13
signal is never exercised; `|tau|` reads < 0.20 even if equalization were
broken. Per scope Q7.8, **hold the SM4+MAC key FIXED and class-split by
PADDING LENGTH**: `Class::Left` = a valid record with a SHORT recovered
fragment (large pad), `Class::Right` = a valid record with a LONG fragment
(minimal pad), **same `body_len`**, both reaching the equalized MAC + CT
compare. Build C1 first; use it to confirm the dummy-compression window before
locking the constant.

**SF1 — `black_box` the dummy-compress per iteration** (not just the trip
count; the loop-invariant `scratch`/`block` can be hoisted/vectorized):
```rust
for _ in 0..n {
    core::hint::black_box(&mut scratch);
    crate::sm3::compress(&mut scratch, core::hint::black_box(&block));
}
let _ = core::hint::black_box(&scratch);
```

**SF2 — `extract_mac_ct` concrete (no secret-offset slicing).** Scan the
PUBLIC range and select (execution-proven byte-exact):
```rust
fn extract_mac_ct(body: &[u8], plaintext_len: usize) -> [u8; 32] {
    let n = body.len();
    let mut out = [0u8; 32];
    for i in 0..=(n - 32) {                              // PUBLIC bound, never eff_len
        let hit = (i as u64).ct_eq(&(plaintext_len as u64));
        for j in 0..32 { out[j] = u8::conditional_select(&out[j], &body[i + j], hit); }
    }
    out
}
```
Unit-test byte-equality at offsets {0, `max_plaintext_len`, mid}; confirm
`|tau| < 0.20` under C1; **do not merge B4 until it exists.**

**SF3 — clippy allows.** `check_tls_padding_ct` + the MF2 casts trip
`-D warnings` (`cast_possible_truncation`/`cast_lossless`). Carry the same
`#[allow(...)]` annotations `mode_cbc::strip_pkcs7_ct` uses, or keep the pad
math in u16/u32 throughout.

**SF4 — fuzz reference body.** The `RecordKeysCbc { mac_key, enc_key }` literal
(C2) can't construct `pub(crate)` fields from the fuzz crate; carve via
`RecordKeysCbc::client_half(&kb)` with layout `[kb:128][seq:8][record:rest]`
(the plan's follow-up note — use it, not the broken literal).

## Wire reference (from `docs/v1.7-scope.md`, RFC-verified)

```
CBC record body  = explicit_IV(16) ‖ CBC_enc( plaintext ‖ MAC(32) ‖ tls_pad )
  tls_pad        = (padlen+1) bytes, each == padlen        (0 ≤ padlen ≤ 255)
  MAC            = HMAC-SM3( mac_key, seq(8)‖type(1)‖ver(2)‖len(2)‖plaintext )
  len            = plaintext length (protect: plaintext.len(); deprotect: recomputed internally — NEVER a param)
GCM record body  = explicit_nonce(8) ‖ ciphertext ‖ tag(16)
  nonce          = salt(4) ‖ explicit_nonce(8)             (explicit = seq, big-endian)
  AAD            = seq(8)‖type(1)‖ver(2)‖len(2)            (len = ciphertext length)
key_block carve (server-first): client_MAC(32) server_MAC(32) client_key(16) server_key(16) [CBC also: client_IV(16) server_IV(16)]
```

Helper, used by both suites:

```rust
/// 13-byte TLS MAC/AAD header. `len` is the PLAINTEXT length.
fn record_header(seq: u64, content_type: u8, version: [u8; 2], len: u16) -> [u8; 13] {
    let mut h = [0u8; 13];
    h[0..8].copy_from_slice(&seq.to_be_bytes());
    h[8] = content_type;
    h[9..11].copy_from_slice(&version);
    h[11..13].copy_from_slice(&len.to_be_bytes());
    h
}
```

use crate::sm4::cipher::Sm4Cipher;
use alloc::vec::Vec;
use zeroize::{Zeroize, ZeroizeOnDrop};

/// TLCP record version on the wire (GB/T 38636: TLS-1.1-style 0x0101).
pub const TLCP_RECORD_VERSION: [u8; 2] = [0x01, 0x01];

/// One direction's CBC record keys, carved from the key block.
/// The IV is NOT here — TLCP CBC uses a fresh RNG-injected explicit IV
/// per record (Q7.6); the key-block IV bytes are vestigial.
#[derive(Clone, ZeroizeOnDrop)]
pub struct RecordKeysCbc { pub(crate) mac_key: [u8; 32], pub(crate) enc_key: [u8; 16] }

/// One direction's GCM record keys (4-byte salt is the implicit nonce half).
#[derive(Clone, ZeroizeOnDrop)]
pub struct RecordKeysGcm { pub(crate) enc_key: [u8; 16], pub(crate) salt: [u8; 4] }

impl RecordKeysCbc {
    /// client-half: client_MAC(0..32) ‖ client_key(64..80).
    #[must_use] pub fn client_half(kb: &[u8; 128]) -> Self {
        let mut mac = [0u8; 32]; let mut enc = [0u8; 16];
        mac.copy_from_slice(&kb[0..32]); enc.copy_from_slice(&kb[64..80]);
        Self { mac_key: mac, enc_key: enc }
    }
    /// server-half: server_MAC(32..64) ‖ server_key(80..96).
    #[must_use] pub fn server_half(kb: &[u8; 128]) -> Self {
        let mut mac = [0u8; 32]; let mut enc = [0u8; 16];
        mac.copy_from_slice(&kb[32..64]); enc.copy_from_slice(&kb[80..96]);
        Self { mac_key: mac, enc_key: enc }
    }
}
impl RecordKeysGcm {
    #[must_use] pub fn client_half(kb: &[u8; 40]) -> Self {
        let mut enc = [0u8; 16]; let mut salt = [0u8; 4];
        enc.copy_from_slice(&kb[0..16]); salt.copy_from_slice(&kb[32..36]);
        Self { enc_key: enc, salt }
    }
    #[must_use] pub fn server_half(kb: &[u8; 40]) -> Self {
        let mut enc = [0u8; 16]; let mut salt = [0u8; 4];
        enc.copy_from_slice(&kb[16..32]); salt.copy_from_slice(&kb[36..40]);
        Self { enc_key: enc, salt }
    }
}
```

use crate::sm4::mode_gcm;

fn gcm_nonce(salt: &[u8; 4], explicit: &[u8; 8]) -> [u8; 12] {
    let mut n = [0u8; 12]; n[0..4].copy_from_slice(salt); n[4..12].copy_from_slice(explicit); n
}

#[must_use]
pub fn protect_gcm(keys: &RecordKeysGcm, seq: u64, content_type: u8, version: [u8; 2], plaintext: &[u8]) -> Option<Vec<u8>> {
    let len = u16::try_from(plaintext.len()).ok()?;          // 2^14 ceiling enforced below
    if plaintext.len() > (1 << 14) { return None; }
    let explicit = seq.to_be_bytes();
    let nonce = gcm_nonce(&keys.salt, &explicit);
    let aad = record_header(seq, content_type, version, len);
    let (ct, tag) = mode_gcm::encrypt(&keys.enc_key, &nonce, &aad, plaintext)?;
    let mut out = Vec::with_capacity(8 + ct.len() + 16);
    out.extend_from_slice(&explicit); out.extend_from_slice(&ct); out.extend_from_slice(&tag);
    Some(out)
}

#[must_use]
pub fn deprotect_gcm(keys: &RecordKeysGcm, seq: u64, content_type: u8, version: [u8; 2], record: &[u8]) -> Option<Vec<u8>> {
    if record.len() < 8 + 16 { return None; }               // need explicit nonce + tag
    let ct_len = record.len() - 8 - 16;
    if ct_len > (1 << 14) { return None; }
    let mut explicit = [0u8; 8]; explicit.copy_from_slice(&record[0..8]);
    let nonce = gcm_nonce(&keys.salt, &explicit);            // explicit nonce from the WIRE (RFC 5288)
    let ct = &record[8..8 + ct_len];
    let mut tag = [0u8; 16]; tag.copy_from_slice(&record[8 + ct_len..]);
    let len = u16::try_from(ct_len).ok()?;
    let aad = record_header(seq, content_type, version, len); // AAD.seq = caller-held seq
    mode_gcm::decrypt(&keys.enc_key, &nonce, &aad, ct, &tag)  // commit-on-verify
}
```

```rust
/// SM3 inner-hash compression-block count for an HMAC over a
/// `plaintext_len`-byte payload: inner input = K'⊕ipad(64) ‖ header(13) ‖
/// plaintext(P); SM3 finalize adds 0x80 + 8-byte length (≥9 bytes).
#[inline]
fn inner_blocks(plaintext_len: usize) -> usize {
    let n = 64 + 13 + plaintext_len;             // pre-pad inner bytes
    n / 64 + 1 + usize::from(n % 64 >= 56)        // +1 final block; +1 if 0x80/len spills
}

/// HMAC-SM3 over `header ‖ plaintext`, with the inner-hash compression
/// count EQUALIZED (via dummy compressions on a throwaway state) to the
/// count for `max_plaintext_len` — a public SAFE UPPER BOUND that MUST NOT
/// be reduced toward a secret value. The real MAC value is unchanged
/// (true length); only the timing is equalized. (Q7.3 surface #1.)
pub(crate) fn mac_equalized(mac_key: &[u8; 32], header: &[u8; 13], plaintext: &[u8], max_plaintext_len: usize) -> [u8; 32] {
    let mut h = crate::hmac::HmacSm3::new(mac_key);
    h.update(header);
    h.update(plaintext);
    let tag = h.finalize();                       // real MAC, true length

    let actual = inner_blocks(plaintext.len());
    let target = inner_blocks(max_plaintext_len);
    let dummy = target.saturating_sub(actual);    // ≥ 0 (max ≥ actual)
    let mut scratch = [0u32; 8];
    let block = [0u8; 64];
    let n = core::hint::black_box(dummy);
    for _ in 0..n { crate::sm3::compress(&mut scratch, &block); }
    let _ = core::hint::black_box(&scratch);      // anti-elision (Q7.3)
    tag
}
```

```rust
use subtle::{Choice, ConditionallySelectable, ConstantTimeEq, ConstantTimeGreater};

/// CT TLS pad check over `body = plaintext ‖ MAC(32) ‖ pad`. Scans a fixed
/// window of min(256, body.len()) trailing bytes — no early return, no
/// padlen-bounded loop. Returns (pad_ok, plaintext_len_if_ok). `plaintext_len`
/// is meaningful only when pad_ok; on !ok the caller uses the public
/// fallback. (Q7.3 surface #2.)
fn check_tls_padding_ct(body: &[u8]) -> (Choice, usize) {
    let n = body.len();                                   // public
    let padlen = body[n - 1];                             // secret
    let pad_region = padlen as u16 + 1;                   // bytes that must all == padlen
    let window = core::cmp::min(256usize, n);
    let mut bad: u8 = 0;
    for i in 0..window {                                  // i = 0 is the last byte
        let pos = (i as u16) + 1;                         // 1..=window
        let byte = body[n - 1 - i];
        let in_pad = !pos.ct_gt(&pad_region);             // pos <= pad_region
        let diff = byte ^ padlen;
        bad |= u8::conditional_select(&0u8, &diff, in_pad);
    }
    // pad_region must leave room for the 32-byte MAC: padlen+1 ≤ n-32  (CT: padlen secret)
    let region_ok = !pad_region.ct_gt(&((n - 32) as u16));
    let ok = region_ok & bad.ct_eq(&0u8);
    // plaintext_len = n - 32 - pad_region, valid only when ok (else 0 — caller ignores)
    let pl = (n as i64) - 32 - pad_region as i64;
    let plaintext_len = usize::conditional_select(&0usize, &(pl.max(0) as usize), ok);
    (ok, plaintext_len)
}
```

```rust
use rand_core::TryCryptoRng;

#[must_use]
pub fn protect_cbc<R: TryCryptoRng + ?Sized>(keys: &RecordKeysCbc, seq: u64, content_type: u8, version: [u8; 2], plaintext: &[u8], rng: &mut R) -> Option<Vec<u8>> {
    let len = u16::try_from(plaintext.len()).ok()?;
    if plaintext.len() > (1 << 14) { return None; }
    let hdr = record_header(seq, content_type, version, len);
    // MAC over the real plaintext (protect side: length is public here).
    let mut h = crate::hmac::HmacSm3::new(&keys.mac_key);
    h.update(&hdr); h.update(plaintext);
    let mac = h.finalize();
    // plaintext ‖ MAC, then TLS pad to a 16-multiple.
    let mut buf = Vec::with_capacity(plaintext.len() + 32 + 16);
    buf.extend_from_slice(plaintext); buf.extend_from_slice(&mac);
    let padlen = 15 - (buf.len() % 16);                   // 0..=15: padlen pad bytes + 1 len byte fills the block
    for _ in 0..=padlen { buf.push(padlen as u8); }       // padlen+1 bytes each == padlen
    // explicit IV (RNG-injected, unpredictable — NIST SP 800-38A).
    let mut iv = [0u8; 16];
    rng.try_fill_bytes(&mut iv).ok()?;
    // raw CBC encrypt (no PKCS#7) over the block primitive.
    let cipher = Sm4Cipher::new(&keys.enc_key);
    let mut prev = iv;
    for chunk in buf.chunks_exact_mut(16) {
        let block: &mut [u8; 16] = chunk.try_into().ok()?;
        for j in 0..16 { block[j] ^= prev[j]; }
        cipher.encrypt_block(block);
        prev = *block;
    }
    let mut out = Vec::with_capacity(16 + buf.len());
    out.extend_from_slice(&iv); out.extend_from_slice(&buf);
    Some(out)
}
```

```rust
#[must_use]
pub fn deprotect_cbc(keys: &RecordKeysCbc, seq: u64, content_type: u8, version: [u8; 2], record: &[u8]) -> Option<Vec<u8>> {
    // Public-length guards (no secret) — BEFORE any secret arithmetic.
    if record.len() < 16 + 48 { return None; }            // IV + min body (0pt+32mac+16pad)
    let body_len = record.len() - 16;
    if body_len % 16 != 0 { return None; }
    let mut iv = [0u8; 16]; iv.copy_from_slice(&record[0..16]);
    let mut body = record[16..].to_vec();
    // raw CBC decrypt (no unpad).
    let cipher = Sm4Cipher::new(&keys.enc_key);
    let mut prev = iv;
    for chunk in body.chunks_exact_mut(16) {
        let block: &mut [u8; 16] = chunk.try_into().ok()?;
        let saved = *block;
        cipher.decrypt_block(block);
        for j in 0..16 { block[j] ^= prev[j]; }
        prev = saved;
    }
    // Surface #2: CT pad check (fixed 256-byte window).
    let (pad_ok, plaintext_len) = check_tls_padding_ct(&body);
    // Public safe upper bound for equalization (#1) — NEVER reduced.
    let max_plaintext_len = body_len - 32 - 1;
    // On bad pad, MAC still runs over a fixed fallback length (never short-circuit).
    let eff_len = usize::conditional_select(&max_plaintext_len, &plaintext_len, pad_ok);
    let len16 = u16::try_from(eff_len).ok()?;             // eff_len ≤ body_len, fits
    let hdr = record_header(seq, content_type, version, len16);
    // The recovered plaintext + received MAC live in `body[..body_len-32-padregion]` /
    // `body[plaintext_len..plaintext_len+32]`. Both offsets are secret → surface #3:
    // extract the 32-byte MAC via a CT scan, and MAC over body[..eff_len] with the
    // equalized compression count (#1).
    let computed = mac_equalized(&keys.mac_key, &hdr, &body[..eff_len], max_plaintext_len);
    let received = extract_mac_ct(&body, eff_len);        // 32 bytes at the secret offset, CT
    let mac_ok = computed.ct_eq(&received);
    // ONE final merge — no branch before here on a secret.
    let valid = pad_ok & mac_ok;
    if bool::from(valid) {
        body.truncate(eff_len);
        Some(body)
    } else {
        body.zeroize();                                   // no plaintext on failure
        None
    }
}
```
> `extract_mac_ct(body, plaintext_len) -> [u8;32]` copies the 32 MAC bytes
> starting at the secret offset `plaintext_len` via a constant-time scan over
> the candidate range (the `strip_pkcs7_ct` mask idiom over the MAC window) —
> surface #3. Implement + unit-test it alongside.

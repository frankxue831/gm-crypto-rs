# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1][cc] as its code
of conduct. The full text is available at:

<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

In short: be respectful and constructive. Harassment, personal attacks, and
discriminatory behaviour are not welcome in issues, pull requests, or any
other project space.

## Reporting

This is a single-maintainer personal project. To report a code-of-conduct
concern privately, open a [GitHub Security Advisory][advisory] (it is a
private channel to the maintainer) or contact the maintainer
[@frankxue831](https://github.com/frankxue831) through GitHub. Reports are
handled confidentially and on a best-effort timeline.

[cc]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
[advisory]: https://github.com/frankxue831/gm-crypto-rs/security/advisories/new
